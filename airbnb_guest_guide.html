Before You Check Out
Quick reset

When you’re ready to check out, a few quick things help us turn the home over smoothly for the next guest:

  • Towels: please leave all used towels in the upstairs bathroom.
  • Bedding: please pull back the comforter on any bed that was used so we know which sheets to replace.
  • Trash: empty all garbage cans and place the tied bags in the garbage cans in the garage.
  • Dishes: place any dirty dishes in the dishwasher and start a cycle if you’re able.

No need to deep clean — just these quick resets are perfect.
